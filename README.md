# 🐾 Сайт ветеринарной клиники

Веб-приложение для ветеринарной клиники с пользовательской частью и админ-панелью для управления контентом.

## 🔗 Демо

https://zoo-clinic.vercel.app

https://github.com/KatrinCl/zoo-clinic

## 🚀 Возможности

### 👨‍⚕️ Пользовательская часть

* Просмотр услуг клиники
* Прайс-лист
* Список врачей с описанием
* Фотогалерея
* Адаптивный дизайн (мобильные / планшеты / ПК)

### 🔐 Админ-панель

* Добавление и удаление врачей
* Добавление и удаление услуг
* Загрузка изображений (через ImageKit)
* Простое управление контентом

## 🛠 Стек технологий

### Frontend

* React (JSX)
* Tailwind CSS
* Axios

### Backend

* Node.js
* Express
* MongoDB (Mongoose)

### Дополнительно

* ImageKit — хранение изображений
* Multer — загрузка файлов

## 📸 Работа с изображениями

* Файлы загружаются через Multer
* Сохраняются в ImageKit
* Используются CDN-ссылки для быстрого отображения

## 📂 Структура проекта

/client — фронтенд (React)
/server — бэкенд (Express API)
/admin - админка (React)

## 🎯 Цель проекта

Проект создан в учебных целях для отработки:

* Fullstack-разработки
* Работы с REST API
* Реализации админ-панели

## 📌 Возможные улучшения
